Apartment Flow

A modern fullstack application built with Node.js/Express backend and Next.js frontend for searching and organizing apartment listings. Features neighborhood information, favorite listings management, and comprehensive CRUD operations.

Project Overview

Apartment Flow is a comprehensive apartment search and management platform that demonstrates advanced full-stack development skills. The application features a modern React frontend with Next.js and a robust Express.js backend with MongoDB integration.

Note: The live demo requires Vercel authentication. For the best experience, explore the GitHub repository to see the complete codebase and features.

Technical Implementation

Frontend (Next.js)

  • Next.js 15 with App Router and React 19
  • TypeScript for type-safe development
  • Responsive design with Tailwind CSS
  • Component-based architecture
  • State management with React hooks
  • Form handling and validation

Backend (Express.js)

  • Express.js with MVC architecture
  • MongoDB with Mongoose ODM
  • JWT-based authentication system
  • RESTful API design
  • File upload with Cloudinary
  • Comprehensive error handling
