FriendBay

A modern social media platform built with Next.js, React, and TypeScript. Features local-only data persistence for privacy, allowing users to create, edit, and interact with posts stored locally on their device.

Project Overview

FriendBay is a privacy-focused social media platform. All posts and data are stored locally on the user's device, ensuring complete privacy. The app demonstrates modern React and Next.js practices, TypeScript usage, and responsive design.

Key Technologies

React
Next.js
TypeScript
Tailwind CSS
Local Storage

Key Features

Privacy & Data

  • Local-only data persistence for complete privacy
  • All posts and user data stored on device

Modern UI & Features

  • Create, edit, and delete posts
  • Responsive, modern UI with Tailwind CSS
  • TypeScript for type safety and maintainability
  • Modern React and Next.js best practices
